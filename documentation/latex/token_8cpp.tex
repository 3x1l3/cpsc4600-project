\subsection{Detailed Description}
The Definition(s) for the \hyperlink{classToken}{Token} Object Class. A token object provides three valuable pieces of information. These are, a Type, a Value, and a Lexeme. These are a Type enum, an integer and a string, respectively.

The token object represents a single PL Symbol -\/ this can be up to two characters in a row -\/ or it represents a data type, or an identifier, etc.

The \hyperlink{classToken}{Token} is used primarily by the \hyperlink{classScanner}{Scanner}, and by proxy, by the Administrator in its functions revolving around the scanner.
