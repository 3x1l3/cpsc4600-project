\subsection{Detailed Description}
The specification for the static \hyperlink{classFirst}{First} set-\/collection object. This class is a deceptively simple one. The \hyperlink{classFirst}{First} object is really just a collection of functions within an easily modifiable class structure that returns the \char`\"{}First set\char`\"{} of a given Nonterminal.

Which is to say, the \char`\"{}first\char`\"{} terminals of a given Nonterminal (or, in our case, function) is returned.
