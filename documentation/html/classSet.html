Detailed Description

The set class manages logical sets of lexemetic values.

A SET is an object containing a vector of lexemic strings.

An overarching Set stopset "sts" is "passed" between nonterminal functions and their productions and is used to mediate the Matching; Said stop set contains the valid values for continuing to parse, and a lack of a specific value within it indicates a syntax error.

The set object uses a Vector of strings to store lexemes. The class also provides core functionality (altering, merging, accessing and viewing the set) in a memory-efficient fashion.
