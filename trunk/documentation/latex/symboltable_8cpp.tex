\subsection{Detailed Description}
The formal definition for our \hyperlink{classSymbolTable}{SymbolTable} (symboltable). We create a symbol table using a cascading order of Maps. This table structure object provides a variety of ways to access and alter entries in the table, and to make new ones and verify the integrity of old ones.
