\subsection{Detailed Description}
The main file for running the \hyperlink{classInterpreter}{Interpreter} seperately. This is a standalone version of the interpreter. It impliments the same class, but allow you to compile and run a separate \hyperlink{classInterpreter}{Interpreter} for any given .itp file.

By default, it uses interpInput.itp.

It takes two arguments -\/ the name of the interp file, and the \char`\"{}Step\char`\"{} flag of -\/s.

Step is used for debugging and stepping through the interp functions by the interpreter.
